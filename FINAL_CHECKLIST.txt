📋 ISTRUZIONI RAPIDE
═══════════════════════════════════════════════════════════════════

1. SCARICA TUTTI I FILE DA /mnt/user-data/outputs/

2. CREA STRUTTURA CARTELLE:
   
   mkdir diet-manager
   cd diet-manager
   mkdir src docs data public

3. COPIA I FILE NEI POSTI GIUSTI:

   NELLA ROOT (diet-manager/):
   - index.html
   - package.json
   - vite.config.js
   - .env.example
   - .gitignore
   - README.md

   IN src/:
   - main.jsx (scaricato come "main.jsx")
   - DietManager.jsx

   IN docs/:
   - SETUP_LOCALE.md
   - CONTRIBUIRE.md
   - SCHEMA_PREZZI.md

   IN data/:
   - prezzario_conad.json
   - prezzario_multi_negozi.json

4. SETUP LOCALE:

   npm install
   cp .env.example .env
   # Apri .env e inserisci: VITE_ANTHROPIC_KEY=sk-ant-api03-xxxxx
   npm run dev
   # Apri http://localhost:5173

5. PUSH SU GITHUB:

   git init
   git add .
   git commit -m "Initial commit: Diet Manager"
   git remote add origin https://github.com/TUO_USERNAME/diet-manager.git
   git push -u origin main
